# Matriculacion vehicular

Sistema orientado a objetos que permite registrar dueños, vehículos, generar matrículas y procesar pagos, 
aplicando herencia y polimorfismo.

## Tecnologías
- Java 17
- Apache Maven 3.9.16
- NetBeans IDE 19

## Dependencias
- JUnit Jupiter 5.10.0 — pruebas unitarias
- Log4j 1.2.17 — registro de eventos
- Apache Commons Lang3 3.12.0 — utilidades de texto
